Pick and Place Mobile Manipulator
Project Overview and Hardware Integration
This project involved the development and implementation of a robust pick-and-place pipeline for a multi-degree-of-freedom mobile manipulator. The system integrates a differential-drive Celer mobile base with a 6-DOF robotic arm within a Gazebo simulation environment. By leveraging ROS 2 and Gazebo-ROS bridging, I orchestrated a seamless synergy between the navigation and manipulation layers, enabling the robot to perform complex, collision-aware tasks in dynamic environments like the AWS Hospital scenario. The work encompassed the full development lifecycle, from hardware abstraction via URDF/SRDF configuration to sensor-driven environment awareness using depth cameras and LIDAR point clouds.

Strategic Implementation with MoveIt Task Constructor
The core of the manipulation logic was built using MoveIt 2 and the MoveIt Task Constructor (MTC), providing a highly modular and hierarchical approach to task planning. I engineered a sophisticated MTC pipeline that handles every stage of the pick-and-place operation, including automated grasp candidate generation, collision-scene management, and sequential task execution with robust error handling. By utilizing a hybrid of sampling-based planners (OMPL) for global connectivity and Cartesian solvers for precise end-effector movements, the pipeline ensures smooth, reliable trajectories while managing complex environmental constraints. This implementation demonstrates advanced proficiency in motion planning, Jacobian-based kinematics, and the design of failsafe autonomous robotic workflows.
