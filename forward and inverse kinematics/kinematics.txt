
# 1. Short Portfolio Description

### **Kinematics, Trajectory Planning, and Data-Driven Inverse Kinematics for XArm 5 Lite**

Developed a complete experimental framework for analyzing the **kinematics and motion planning of the XArm 5 Lite 5-DoF robotic manipulator**. The work involved deriving **Denavit–Hartenberg (DH) parameters**, implementing **forward kinematics in MATLAB**, and validating theoretical end-effector positions using the physical robot through **XArm Studio**.

Implemented **joint-space trajectory planning using a trapezoidal velocity profile** and experimentally compared theoretical motion duration with real robot execution times under varying velocity and acceleration parameters.

Additionally, explored **machine learning–based inverse kinematics**, generating training datasets using forward kinematics and evaluating **25 regression algorithms** to predict joint configurations from end-effector coordinates. Results showed that **Gaussian Regression and Support Vector Machines outperformed deep learning approaches** in prediction accuracy.

**Technologies:** MATLAB, ROS, XArm Studio, Machine Learning Regression Models, DH Modeling, Robot Kinematics, Trajectory Planning.

# 2. Detailed Portfolio Description

### **Kinematic Modeling, Trajectory Planning, and Machine Learning–Based Inverse Kinematics for XArm 5 Lite Manipulator**

This project investigates the **kinematic modeling, motion planning, and data-driven inverse kinematics** of the **XArm 5 Lite collaborative robotic manipulator**. The work combines theoretical robotics modeling with experimental validation on a physical robot platform.

---

## **Forward Kinematics and DH Parameter Modeling**

The first phase focused on establishing the **forward kinematic model** of the manipulator using the **Denavit–Hartenberg (DH) convention**. Accurate DH parameters were derived for all five joints of the robot.

Since the mechanical zero configuration of the robot differs from the mathematical reference configuration used in kinematic modeling, **joint offsets (T2_offset, T3_offset, T4_offset)** were introduced to align the physical system with the theoretical model.

Using these parameters, **MATLAB scripts were developed to compute homogeneous transformation matrices** for each joint. The transformation matrices were sequentially multiplied to obtain the final **end-effector pose with respect to the base frame**.

To validate the model, several joint configurations were tested on the **physical XArm 5 Lite robot using XArm Studio**. The experimentally observed end-effector positions were compared with MATLAB calculations to verify the correctness of the derived kinematic model.

---

## **Joint-Space Trajectory Planning**

The second phase investigated **joint-space trajectory generation** using a **trapezoidal velocity profile**, a widely used method for smooth robotic motion.

A motion task was defined where the first joint moved from **30° to 75°**. The motion was divided into three phases:

* Acceleration
* Constant velocity
* Deceleration

Using predefined acceleration and velocity parameters, the **theoretical trajectory duration was calculated as 3.25 seconds**.

The same trajectory was then executed on the physical robot using **block-based programming in XArm Studio**, and the **actual execution time was measured using a stopwatch**. Multiple trials were conducted with different motion parameters to analyze the difference between theoretical predictions and real robot behavior.

This experiment provided practical insight into **robot motion control, actuator limits, and dynamic response of robotic manipulators**.

---

## **Machine Learning–Based Inverse Kinematics**

In addition to classical kinematic methods, this project explored a **data-driven approach for solving inverse kinematics**.

A large dataset was generated by computing forward kinematics for numerous joint configurations using the DH model. The dataset mapped **end-effector Cartesian coordinates to corresponding joint angles**.

Instead of relying solely on neural networks, the study evaluated **25 different machine learning regression algorithms** to determine the most accurate model for predicting joint angles.

Key findings included:

* **Gaussian Regression and Support Vector Machines achieved higher accuracy** than deep learning models.
* Prediction accuracy varied across joints, with **Joint 2 showing the highest reliability**.
* Data-driven models can provide robust solutions for inverse kinematics, particularly in cases where traditional analytical solutions are difficult or prone to singularities.

---

## **Simulation and Development Environment**

To support further experimentation, a **robotics development environment was established using ROS and the xarm_ros packages**. A **Google Colab notebook** was also configured for rapid experimentation and algorithm development.
